Scope Statement

The objective of this project is to create a website which shows each component of the Signature Assignment. Also, this website may be used to disseminate information to future project participants. Planning for this project began on June 11, 2020 and the completion deadline is June 28, 2020. Total costs for this project was $6, which is the cost for registering the domain name. Even though project work was not outsourced to a web developer, the budget table on this website represents what it would cost to outsource project work. Given that project work was not outsourced, implementing the plans contained in the Signature Assignment was achievable.

Significant events for this project include the completion of each project phase. First, completion of phase I is significant because it marks the end of the initiation phase. In the initiation phase, requirements for the website are identified and a general idea of the finished product's purpose has been established. Second, completion of phase II is significant because it marks the end of the building phase. At the end of phase II, the website should be fully functional and only minor changes should be needed. Last, completion of phase III is significant because it marks the end of the quality control phase. When the quality control phase has been completed, the website can go "live."

Third-party software/services:

    1. text editor
    2. cloud-based storage
    3. web hosting service
    4. domain name registrar

The technical requirements for this project can be grouped into two categories. First, knowledge of programming and markup languages such as Python, HTML, CSS, and JS ensures the proper performance of the website. Python is the programming language which handles all back-end operations. HTML, CSS, and JS are front-end languages responsible for the website's appearance. Second, several third-party applications and services are required in order to build and publish a website. Sublime Text has been chosen as the text editor which is the platform where code is written into. GitHub is a cloud-based services which allows programmers to store their projects. Data stored on GitHub can be pushed to a web hosting service for publication such as Heroku. Last, a domain name registrar is required to create a unique website name. Although, there are several domain name registrars to choose from, Namecheap is the domain name registrar chosen for this project.
